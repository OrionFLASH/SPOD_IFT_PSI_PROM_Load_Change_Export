Документ объединяет прежние версии `ROADMAP.md` и `Docs/Roadmap_SPOD.md`.

## Формат статусов

| Маркер | Значение |
|--------|----------|
| `[v]` | Сделано |
| `[w]` | В работе / требует доработки |
| `[ ]` | Не сделано |
| `[x]` | Отменено |

---

## Цель

Привести поведение пайплайна к правилам:
- загружать все строки из всех таблиц и стендов без исключений;
- сохранять дубли (включая дубли по ключу внутри одного файла);
- корректно выбирать строку в отчете по приоритету стендов;
- собирать подробные различия в `DIFF_REPORT`;
- формировать сводные отчеты консистентности (`CONSISTENCY`, `CONS_REPORT`).
