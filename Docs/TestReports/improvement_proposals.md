# Предложения по улучшению (без дополнительных pip-модулей)

Все пункты ниже реализуемы на стандартной библиотеке Python + уже имеющихся пакетах окружения.

## Реализовано в коде

- Параллельная обработка с авто-определением числа потоков по ядрам машины.
- Пакетная запись в SQLite (`executemany`) для ускорения.
- Лист `DIFF_REPORT` в Excel со сводкой конфликтов.
- Профилирование времени этапов (`scan/parse/merge/save/export`) в логах.
- Режим `dry-run` для запуска без сохранения Excel и без записи `raw_rows`/`merged_rows`.
- CLI-управление: `--dry-run`, `--parallel-workers`.
- Параметры контекста `diff_snippets` вынесены в `config.json`.
- Слой **`consistency_checks`**: лист `CONSISTENCY`, правила C-01…C-10, `scope` / `both`, интеграция в пайплайн.
